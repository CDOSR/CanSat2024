# Component Data Sheets

Manufacturer datasheets for the components used on the CanSat 2024
custom PCBs. One PDF per part, filename matches the manufacturer part
number.

| File | Part | Role |
|---|---|---|
| `AP2112M-3.3TRG1.pdf` | Diodes AP2112M-3.3 | 3.3 V LDO regulator |
| `BME280.pdf` | Bosch BME280 | Pressure / humidity / temperature |
| `BME688.pdf` | Bosch BME688 | Environmental + gas sensor |
| `ICM-20648.pdf` | TDK InvenSense ICM-20648 | 6-axis IMU |
| `L76-M33.pdf` | Quectel L76 | GNSS module |
| `LPS25HBTR.pdf` | ST LPS25H | MEMS pressure sensor |
| `MAX17048-MAX17049.pdf` | Maxim MAX17048/049 | Li-ion fuel gauge |
| `MCP73831T-2ACI_OT.pdf` | Microchip MCP73831 | Li-ion charger IC |
| `MCP9808-E_MS.pdf` | Microchip MCP9808 | Digital temperature sensor |
| `RN2483A-I_RM105.pdf` | Microchip RN2483 | LoRaWAN radio module |
| `VEML6075.pdf` | Vishay VEML6075 | UVA / UVB sensor |

The boards that use these parts — schematic and layout — live under
[`Hardware/PCB Designs/`](../../../Hardware/PCB%20Designs/) at the repo
root.
