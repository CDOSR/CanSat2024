# Hardware Documentation

Reference material for the CanSat 2024 hardware subsystems.

## Contents

- `Data Sheets/` — Manufacturer datasheets for every sensor, regulator,
  GNSS module, radio, and battery IC used on the custom boards.

## Related

The team's own hardware sources — Eagle `.sch` and `.brd` CAD files for
the four custom boards — live under [`Hardware/`](../../Hardware/) at
the repo root, so that vendor reference PDFs don't clutter the CAD tree.
